# ✈ CabinLog
O **CabinLog** é um projeto individual desenvolvido para o curso de **Análise e Desenvolvimento de Sistemas (ADS)** da **Faculdade São Paulo Tech School**.

## 🎯 Proposta: 
É uma plataforma web inspirada em redes sociais, voltada para profissionais da aviação compartilharem suas experiências, conquistas e momentos da carreira na aviação por meio de publicações com fotos e descrições.

## 🧠 Tecnologias utilizadas:
- HTML
- CSS
- JavaScript  
- MySQL Workbench  
- Node.js 
- Canva  
- Trello  
- VirtualBox  
- GitHub
- GitBash
